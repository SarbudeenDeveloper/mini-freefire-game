# Mini Free Fire Game

## How to Run

1. Navigate to the `server` directory:
```bash
cd server
```

2. Install the required packages:
```bash
npm install
```

3. Start the server:
```bash
node server.js
```

4. Open the game in your browser at:
```
http://localhost:3000
```
To join more players, open localhost:3000 in multiple tabs.
